# Bleak to WebSocket bridge

Simple bleak remote server over WebSocket

## Run Demo (Bleak scanner server and web client)

> [!TIP] 
> I recomment you to use `uv` package manager

Simplest way
```Console
uvx --from https://github.com/o-murphy/ble2ws.git ble2ws-demo
```

Clone, and sync
```console
git clone https://github.com/o-murphy/ble2ws
cd ble2ws
uv sync
.venv/bin/activate
```
